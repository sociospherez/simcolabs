# SimCo Labs Starter

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Why this setup

This starter uses React + Vite + Tailwind so SimCo Labs can grow from a landing page into a showcase platform for MVPs, concepts, dashboards and experiments.

## Where to add projects

Create a new JSON file in:

`src/content/projects/`

Then import it into:

`src/content/projects/index.js`

## Current routes

- `/`
- `/showcase`
- `/research`
- `/about`
- `/contact`
- `/projects/:slug`
